API — Application Programming Interface (Interfaz de Programación de Aplicaciones)
Es una herramienta que permite que dos programas diferentes se comuniquen entre sí, como si hablaran el mismo idioma.
Funciona como un camarero digital: toma tu pedido (solicitud), lo lleva a la cocina (servidor) y te trae lo que pediste (respuesta), sin que tú veas todo lo que pasa detrás.
SEO — Search Engine Optimization (Optimización para Motores de Búsqueda)
Es el proceso de ajustar tu sitio web para que los motores de búsqueda como Google lo entiendan mejor y lo muestren entre los primeros resultados.
En pocas palabras, es como hacer que tu página sea la respuesta perfecta que Google quiere mostrar.
